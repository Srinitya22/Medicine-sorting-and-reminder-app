# 💊 Medicine Sorting and Reminder App

Nowadays, people of every age group use medicines regularly, and it often becomes difficult to organize them properly within a family. Medicines can get mixed up, dosages may be missed, reminders are forgotten, and managing multiple family members' medicines becomes confusing.

To solve this problem, I developed the **Medicine Sorting and Reminder App**.

This application helps users manage medicines efficiently by allowing them to:

- 📷 Scan medicine labels
- ✍️ Manually enter medicine details
- 🎤 Record medicine names using voice input
- 👤 Add multiple profiles under a single account
- 🔐 Register/Login using Google Authentication or Email
- ⏰ Set and save medicine reminders
- 📅 Track medicine intake using a calendar feature
- ⚠️ Receive warnings when a dosage is missed

The project is designed to make medicine management easier, organized, and user-friendly for families and individuals.

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Service Workers / PWA Features
